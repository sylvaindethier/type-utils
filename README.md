## Install

- `npm`

  ```sh
  npm install -D @sdt/typescript-utils
  ```

- `pnpm`

  ```sh
  pnpm add -D @sdt/typescript-utils
  ```

- `yarn`

  ```sh
  yarn add -D @sdt/typescript-utils
  ```
